## Getting Started

Clone this repository,
``` bash
git clone https://github.com/websemantics/atom-styleguide
```

Install dependencies,
``` bash
cd atom-styleguide
npm install
```

Build or Watch to `dist` folder,
``` bash
npm run build 
npm run watch 
```

Start development server,
``` bash
npm run start 
```
